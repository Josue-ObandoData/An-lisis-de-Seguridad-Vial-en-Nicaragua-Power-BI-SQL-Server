# 🚦 Análisis de Seguridad Vial en Nicaragua | Power BI + SQL Server

Dashboard interactivo desarrollado en Power BI utilizando SQL Server para el modelado y transformación de datos relacionados con accidentes de tránsito en Nicaragua durante el período Enero - Mayo 2026.

---

# 📊 Objetivo del Proyecto

Analizar el comportamiento de los accidentes de tránsito en Nicaragua mediante visualizaciones interactivas que permitan identificar:

- Departamentos con mayor incidencia
- Tendencias mensuales
- Tipos de vehículos más involucrados
- Relación entre regulación policial y accidentes
- Semanas más críticas

---

# 🛠 Herramientas Utilizadas

- Power BI
- SQL Server
- Excel
- Power Query
- DAX

---

---

# 📌 Principales KPIs

- Total de accidentes
- Total de lesionados
- Porcentaje de lesionados
- Pico semanal de accidentes

---

# 📈 Visualizaciones Incluidas

- Tendencia de accidentes por mes
- Top departamentos con más accidentes
- Vehículos con mayor participación en accidentes
- Regulación policial por mes
- Relación entre alcohol y accidentes

---

# 🧠 Insights Obtenidos

- Managua concentra más del 77% de los accidentes registrados.
- Las motocicletas representan el tipo de vehículo con mayor participación en accidentes.
- Se identificaron semanas críticas con altos niveles de incidencia.
- La regulación policial no siempre presenta relación directa con la disminución de accidentes.

---

# 🗂 Modelo de Datos

El proyecto utiliza un enfoque tipo estrella (Star Schema), utilizando una tabla fact principal conectada a tablas dimensionales para permitir escalabilidad y análisis interactivo en Power BI.
